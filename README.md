# 塔漠里健康科技静态官网 Demo

这是一个仅使用 HTML、CSS、JavaScript 的静态企业官网 Demo，包含首页产品展示、产品搜索，以及一个用于演示的后台产品发布页面。

## 如何打开首页

1. 在项目目录中直接用浏览器打开 `index.html`。
2. 也可以使用本地静态服务器预览，例如：

   ```bash
   python3 -m http.server 8000
   ```

   然后访问 `http://localhost:8000/index.html`。

## 如何测试后台新增产品功能

1. 打开 `admin.html`，或在首页点击“后台发布模拟 / 发布新产品”。
2. 在表单中填写：
   - 产品名称
   - 产品分类
   - 产品简介
3. 点击“保存产品”。产品信息会保存到当前浏览器的 `localStorage`。
4. 返回 `index.html`，新增产品会和默认产品一起出现在“产品展示”区域。
5. 在首页搜索框中输入产品名称或分类，可以验证默认产品与后台新增产品都能被筛选出来。

> 注意：`localStorage` 数据保存在当前浏览器中。如果换浏览器、使用隐私模式或清除站点数据，后台新增的产品记录会消失。
